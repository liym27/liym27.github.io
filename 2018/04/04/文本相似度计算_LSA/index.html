文本相似度 之LSA

发表于 2018-04-04 | 阅读次数:

先说向量空间模型（VSM）

  VSM的思想是把文档简化为以特征项（关键词）的权重为分量的N维向量表示。这个模型属于词袋模型，文本被看做是无序的词汇集合，忽略语法甚至是单词的顺序。VSM模型的缺点在于关键词之间的线性无关的假说前提，这个前提造成VSM模型无法进行语义相关的判断。但VSM使用向量来表示文本，简化了文本中的关键词之间的复杂关系，文档用十分简单的向量表示，使得模型具备可算性。
  BM25就是一种VSM模型。

LSA算法

原理简述

 VSM将各关键词之间假设为线性无关的这个前提造成VSM模型无法进行语义相关的判断，没有能力处理更复杂的一词多义、一义多词问题。而潜在语义分析（Latent Semantic Anaiusis,LSA）是一种自动实现知识提取和表示的理论和方法。与VSM类似，都是采用空间表示文本，但是通过SVD等处理，一定程度上解决了一词多义、一义多词的影响，提高了后续处理的精度。

def get_lsi(corpus_tfidf):
    lsi_model = models.LsiModel(corpus_tfidf,num_topics=4000)
    corpus_lsi = lsi_model[corpus_tfidf]
    similarity_lsi = similarities.Similarity('Similarity-LSI-index', corpus_lsi, num_features=4000,num_best=10)
    return lsi_model, similarity_lsi
