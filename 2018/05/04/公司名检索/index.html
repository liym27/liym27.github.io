1.2 query likelihood(QL)

语言模型的假设：文档d与查询q相关的概率 约等于 在文档相关的前提下用户输入q的概率。
$ relevance(q,a) = P(R=1|q,d) => p(q|d,R=1) $
query likelihood ranking 函数：$f(q,d) = p(q|d)$
表示搜索 document d 的用户输入 query q 的概率。

1.2.1 概率模型基本概念

$$ f(q,d) = p(R=1|q,d) = \frac{count(q,d,R=1)}{count(q,d)} $$
问题：对于数据集中没有的query，无法获取概率值。
